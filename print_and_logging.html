Функция print и логирование в python

Всем привет! В этой статье мы позна­комимся со стандартным потоком вывода, различными вариантами вывода контента в файлы и разберемся с понятием "логирование" в языке программирования python. Устраивайтесь по-удобнее, будет интересно!

Функция print

Функция print выводит переданные в неё аргументы в стандартный поток вывода. Что же такое стандартный поток вывода? Standart output или stdout называется потоком вывода, местом, куда мы выводим наш текстовый контент. По умолчанию стандартный поток вывода равен sys.stdout и поэтому вывод осуществляется в консоль.

Функция print все переданные в неё аргументы в стандартный поток вывода. Например:

print(1)
print('Hello world!')
print(False)
print(1.5, 2.0, 10, True, 'username')
print('Hello' + ' ' + 'world' + '!')
# 1
# Hello world!
# False
# 1.5 2.0 10 True username
# Hello world!

На этом тривиальном примере мы видим две вещи. Во-первых, каждый print переносит поток вывода на новую строчку. Во-вторых, аргументы, переданные в функцию print выводятся через пробел.

Во-первых: print(1)
print(2)
# 1
# 2

Во-вторых:
print('Hello', 'world')
# Hello world

В обоих случаях мы можем изменить стандартное поведение. Рассмотрим первый параметр функции print — end, в него передается строка, которая напечатается после всех аргументов функции print.

print(1, end=' ')
print(2, end=' ')
print(3, end=' ')
# 1 2 3
print(1, end=''-'')
print(2, end='-')
print(3, end=''-'')
# 1-2-3-
print(1, end='-я выведусь после первого print-')
print(2, end='-a я после второго-')
print(3, end='-я выведусь после третьего-')
# 1-я выведусь после первого print-2-a я после второго-3-я выведусь после третьего-

Рассмотрим второй параметр функции print — sep, sep от английского separator (разделитель). По умолчанию параметр sep равен ' '. Время для экспериментов ╰(▔∀▔)╯.

print(1, 2, 3, 4)
print(1, 2, 3, 4, sep='+++')
print(1, 2, 3, 4, sep='разделитель')
print(1, 2, 3, 4, sep='(◕‿◕)')
print(1, 2, 3, 4, sep='(ﾉ◕ヮ◕)ﾉ*:･ﾟ✧')
# 1 2 3 4
# 1+++2+++3+++4
# 1разделитель2разделитель3разделитель4
# 1(◕‿◕)2(◕‿◕)3(◕‿◕)4
# 1(ﾉ◕ヮ◕)ﾉ*:･ﾟ✧2(ﾉ◕ヮ◕)ﾉ*:･ﾟ✧3(ﾉ◕ヮ◕)ﾉ*:･ﾟ✧4

Логирование в python

Поговорим немного о логировании. Логирование - это создание логов работы нашей программы. Что же такое лог? Лог - это текстовый файл, в который записывается промежуточные результаты работы нашей программы. С помощью логов мы можем отслеживать выполнение нашей программы и, следовательно, оперативно устранять ошибки, возникающие в процессе написания кода. Вы уже пользовались логированием. Обычный вывод текстового контента в стандартный поток вывода (в нашем случае, в консоль) помогает нам отслеживать этапы выполнения нашего кода. Напишем небольшой самодельный скрипт-логгер, который генерирует рандомную матрицу (список списков) целых чисел и выводит значения этой матрицы и в консоль, и в файл log.txt:

from random import randint

some_list = [[randint(0, 100) for j in range(15)] for i in range(15)]

with open('log.txt', 'w') as log_file:
    for inner_list in some_list:
        for element in inner_list:
            print(element, end=' ', file=log_file)
            print(element, end=' ')
        print()
        print(file=log_file)

Функция print имеет еще один дополнительный параметр - file. Он определяет место вывода текстового содержимого нашей программы.

Модуль pprint

Модуль pprint (pretty print) выполняет "красивую" печать в указанный поток вывода. В отличии от обычной функции print, pprint принимает несколько дополнительных аргументов: width - количество знаков в одной линии выводимого контента, indent - величина отступа, depth - глубина вложенности, compact - плотность вывода (определяет полностью ли мы заполним нашу width вывода), а также поток вывода, куда будем выводить контент.

from pprint import pprint
from random import randint

some_list = [[randint(0, 100) for j in range(10)] for i in range(10)]
            
pprint(some_list)
            
print(some_list)

some_list = [[randint(0, 100) for j in range(15)] for i in range(10)]

with open('log.txt', 'w') as log_file:
    pprint(some_list, stream=log_file)
    pprint(some_list)
