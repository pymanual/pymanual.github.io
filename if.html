Условный оператор

Что такое условный оператор и с чем его едят?

Всем привет! В этой статье мы позна­комимся с такой чудесной штукой, как условный оператор if...else. Представим себе такую ситуацию:

Вы сидите дома и хотите пойти погулять. Будучи разумным человеком вы решаете перед выходом по­смот­реть прогноз по­го­ды, чтобы не прогадать с выбором предметов одеж­ды. Погодная сводка пу­гает вас дождем и по­ры­ва­ми ветра до 20 метров в секунду. Вы решаете надеть куртку и взять с собой зонт.

В примере выше, вы решаете, какую одежду стоит надеть, основываясь на внешних факторах. если на улице холодно и идет дождь, то я надену теплую одежду и возьму зонт, иначе сделаю выбор в пользу футболки и новеньких шорт.

Итак, переходим к примерам:

# isRaining равна True, значит идет дождь
isRaining = True
if isRaining:
    print('Надеть теплую одежду')
else:
    print('Ура! Футболка и шорты!')
# Надеть теплую одежду
# isRaining равна False, значит на улице солнышко!
isRaining = False
if isRaining:
    print('Надеть теплую одежду')
else:
    print('Ура! Футболка и шорты!')
# Ура! Футболка и шорты!

Это и есть условный оператор if...else. Работает это так: если выражение после if равно True, то будет выполнен блок код, на­ходя­щий­ся под if

if True:
    print('Я напечатаюсь')
else:
    print('А я - нет :(')
# Я напечатаюсь

В противном случае, если выражение под if равно False, то выполнится блок кода, на­ходя­щий­ся под else

if False:
    print('Эй, что за дела?')
else:
    print('АХАХАХ, ну наконец-то')
# АХАХАХ, ну наконец-то

Иногда достаточно только блока if

if True:
    print('Одному как-то одиноко...')
# Одному как-то одиноко...
# Здесь ничего не выведется,
# потому что функция print не будет вызвана
if False:
    print('Так, подожди, в каком смысле False?')
#

Было у царя три сына...

У if и else есть братик - elif. Всё сложнее и сложнее, зачем нам нужен elif? Конструкция if...else пред­став­ляет собой логические выражения вида:

если условие истинно, то делаем действие1, иначе действие2

А что если у нас несколько ва­риан­тов? Для этого и нужен elif

если первое условие истинно, то делаем действие1,
если второе условие истинно, то делаем действие2,
если третье условие истинно, то делаем действие3,
иначе действие4

Приведу простой пример:

number = 10
if number == 3:
    print('number равен 3')
elif number == 5:
    print('number равен 5')
elif number == 10:
    print('number равен 10')
else:
    print('Оно тебе надо? Держи лучше котика ≧(◕ ‿‿ ◕)≦')
# number равен 10

Мы не ограничены в количестве блоков elif

number = 'А я вообще строка ( ͡° ͜ʖ ͡°)'
if number == 3:
    print('number равен 3')
elif number == 5:
    print('number равен 5')
elif number == 10:
    print('number равен 10')
elif number == 100:
    print('number равен 100')
elif number == 999:
    print('number равен 999')
else:
    print('Оно тебе надо? Держи лучше котика ≧(◕ ‿‿ ◕)≦')
# Оно тебе надо? Держи лучше котика ≧(◕ ‿‿ ◕)≦

Можно и без else

mood = 'happy'
if mood == 'happy':
    print('(◕‿◕)♡')
elif mood == 'sad':
    print('Не грусти (づ｡◕‿‿◕｡)づ')
# (◕‿◕)♡

Коротенькие примерчики

Напишем небольшой скрипт, который определяет является ли вве­ден­ное число четным:

number = int(input('Введите число: '))
if number % 2 == 0:
    print('Вы ввели четное число')
else:
    print('Вы ввели нечетное число')

Еще один примерчик: програм­ма, опреде­ляющая является ли введен­ное число поло­житель­ным:

number = int(input('Введите число: '))
if number > 0:
    print('Вы ввели положительное число')
elif number == 0:
    print('Вы ввели ноль!')
else:
    print('Вы ввели отрицательное число')
